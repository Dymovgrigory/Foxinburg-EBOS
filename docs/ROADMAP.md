## ✅ Фаза 0: Фундамент (завершено)

- [x] Брендовые стили: `fox-purple: #3A2953`, `fox-gold: #F5ED75`, Tailwind-конфиг.
- [x] `AuthContext` на frontend, `localStorage` для токена, ролевая маршрутизация.
- [x] Pydantic-схемы для auth, единый формат `{success, message, data}`.
- [x] RBAC: 9 ролей, `Permission`, `require_role`, `has_permission`.

- [x] PostgreSQL + Redis + async SQLAlchemy.
