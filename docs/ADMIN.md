# Админ-панель FOXINBURG

Админ-панель построена на [SQLAdmin](https://github.com/aminalaee/sqladmin) — Django-like интерфейс для FastAPI/SQLAlchemy.

## URL

```
https://foxinburg.ru/admin
```

## Доступ

Вход доступен только пользователям с ролью:

- `OWNER`
- `SUPER_ADMIN`
- `ADMIN`

Студенты, преподаватели и другие роли не могут войти.

## Возможности

Полный CRUD для всех сущностей системы:

- Пользователи, организации, филиалы
- Курсы, модули, уроки, контент
- Тесты, вопросы, попытки
- Группы, зачисления, прогресс
- Домашние задания и проверки
- CRM (лиды, сделки)
- Финансы (платежи, транзакции)
- Расписание и посещаемость
- Уведомления, ачивки, файлы
- Чаты и сообщения
- Системные события и audit-логи

## Вход

1. Открой `https://foxinburg.ru/admin`.
2. Введи email и password администратора.
3. После успешного входа откроется список разделов.

## Выход

Нажми **Logout** в правом верхнем углу или перейди по ссылке:

```
https://foxinburg.ru/admin/logout
```

## Безопасность

- Аутентификация через JWT-токен, хранимый в сессии.
- Токен проверяется при каждом запросе; заблокированные пользователи теряют доступ.
- Доступ разрешён только ролям OWNER/SUPER_ADMIN/ADMIN.
